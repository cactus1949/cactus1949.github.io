常用markdown语法一览

记录一些常用的markdown文档书写格式，以便于博客的书写，有使用到的新的语法即会更新此博客。

标题

# h1
## h2
...
###### h6

列表

  1. 无序列表
    ```
  • list
  • list
    ```
  1. 有序列表
    ```
  2. xxx
  3. xxx
    ## 多选框
  • [] xxx
    ## 超链接
    xxx
    ```

    图片

  1. 可以使用html进行插入图片，附加内嵌css进行修饰

    <img src="" style="" alt=""></ img>

  2. md格式

    {% image 图片链接 '文字说明' '' %}

    引用

    > 这是一段引用

    代码段

    ```

    这里是一段代码

    这里是一段代码

    ```
